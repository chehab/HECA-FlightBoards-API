HECA/CAI FlightBoard API
v2.1 Beta

Access Cairo Int. Airport Flight Board
for Arrival, Departure or Both through JSON or XML formats




  • GET Request:
    • Request Return Type JSON or XML
      • [ json | xml ] = [ true | 1 ]
      • type = [ json | xml ]
    • Request either Arrival, Departure or Both
      • [ arrival | departure | both ] = [ true | 1 ]
      • flights= [ arrival | departure | both ]
    • Request Example
      • /?type=json&arrival=1
      • /?json=true&flights=arrival

  • Upcoming:
    • IATA, ICAO Airports Codes
    • Airports Information
    • Get updated flights only
    • Get flights within a time range
    • Flight Search via flight#, distention, origin and Airline
